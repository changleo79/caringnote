# 📝 마이그레이션 수동 실행 안내

## 🔍 현재 상황

현재 터미널에서 Node.js/npm 명령어를 찾을 수 없습니다. 다음 방법 중 하나로 실행하세요.

---

## ✅ 방법 1: 다른 터미널에서 실행

### VS Code 통합 터미널 사용

1. **VS Code 열기**
2. **터미널 메뉴** → **새 터미널** (Ctrl + Shift + `)
3. 다음 명령어 실행:

```powershell
# 1. Prisma 클라이언트 생성
npm run db:generate

# 2. 데이터베이스에 적용
npm run db:push
```

---

## ✅ 방법 2: 명령 프롬프트(CMD) 또는 PowerShell에서 실행

1. **Windows 키 + R** 누르기
2. `cmd` 또는 `powershell` 입력 후 Enter
3. 프로젝트 폴더로 이동:
   ```cmd
   cd "C:\Users\82103\Desktop\cursor\care app"
   ```
4. 명령어 실행:
   ```cmd
   npm run db:generate
   npm run db:push
   ```

---

## ✅ 방법 3: 배치 파일 더블클릭

1. **파일 탐색기** 열기
2. 프로젝트 폴더로 이동: `C:\Users\82103\Desktop\cursor\care app`
3. **`run_migration.bat`** 파일을 더블클릭
4. 자동으로 마이그레이션이 실행됩니다

---

## ✅ 방법 4: Vercel 배포 시 자동 실행

Vercel에 배포하면 `postinstall` 스크립트가 자동으로 `prisma generate`를 실행합니다.

배포 후 Vercel에서:
1. **Settings** → **Environment Variables** 확인
2. **Deployments** 탭에서 배포 확인
3. 배포 완료 후 데이터베이스 연결 확인

---

## 📋 실행해야 할 명령어

### 개발 환경
```bash
# 1. Prisma 클라이언트 생성
npm run db:generate

# 2. 데이터베이스 스키마 적용
npm run db:push
```

### 프로덕션 환경 (Vercel)
```bash
# Vercel은 자동으로 실행하지만, 수동으로 필요하면:
npx prisma generate
npx prisma db push
```

## 🔍 실행 가능 여부 확인

실행 전에 다음을 확인하세요:

```powershell
npm --version

# Prisma 확인
npx prisma --version
```

## ❓ Node.js가 설치되어 있지 않은 경우

### Node.js 설치 방법

1. **Node.js 공식 사이트 접속**
   - https://nodejs.org/

2. **LTS 버전 다운로드 및 설치**
   - Windows Installer (.msi) 다운로드
   - 설치 시 "Add to PATH" 옵션 체크

3. **설치 확인**
   ```powershell
   node --version
   npm --version
   ```

4. **프로젝트 폴더에서 실행**
   ```powershell
   npm run db:generate
   npm run db:push
   ```
