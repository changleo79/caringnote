# 프로젝트 개발 가이드

## 📋 프로젝트 개요

키즈노트 스타일을 참고한 요양원 케어 플랫폼으로, 다음 기능을 제공합니다:

1. **사진 공유 및 커뮤니티**: 요양원에서 부모님의 일상을 사진으로 공유하고 가족들과 소통
2. **의료 정보 공유**: 건강 상태, 진료 기록, 약물 복용 정보 등을 투명하게 공유
3. **생필품 쇼핑몰**: 필요한 생필품을 구매하고 요양원으로 직접 배송
4. **회원 관리**: 요양원 회원/일반 회원 구분, 요양원 선택 기반 가입

## 🏗️ 현재 완성된 구조

### ✅ 완료된 항목

1. **프로젝트 초기 설정**
   - Next.js 14 (App Router)
   - TypeScript
   - Tailwind CSS
   - Prisma ORM
   - NextAuth.js 인증 시스템

2. **데이터베이스 스키마**
   - 요양원 (CareCenter)
   - 사용자 (User) - 가족/요양원 직원 구분
   - 입소자 (Resident)
   - 게시글 (Post) - 커뮤니티
   - 의료 기록 (MedicalRecord)
   - 상품 및 주문 (Product, Order)
   - 댓글 및 좋아요 (Comment, PostLike)

3. **인증 시스템**
   - 회원가입 (요양원 선택 포함)
   - 로그인
   - 세션 관리
   - 역할 기반 접근 제어

4. **기본 UI/UX**
   - 반응형 레이아웃 (모바일/데스크톱)
   - 하단 네비게이션 (모바일)
   - 사이드바 (데스크톱)
   - 대시보드

5. **배포 준비**
   - Vercel 설정
   - GitHub 연동 가이드
   - 환경 변수 관리

## 🎨 키즈노트 스타일 참고 사항

키즈노트의 성공 요소를 반영:

1. **직관적인 UI**: 복잡하지 않고 사용하기 쉬운 인터페이스
2. **실시간 알림**: 중요한 정보를 빠르게 전달
3. **사진 중심**: 시각적 소통 강조
4. **간편한 가입**: 요양원 선택으로 쉽게 시작
5. **신뢰성**: 투명한 정보 공유로 신뢰 구축

## 📱 웹 → 앱 전환 전략

현재 웹 기반 프로토타입으로 개발 중이며, 이후 다음과 같은 옵션을 고려할 수 있습니다:

1. **PWA (Progressive Web App)**
   - 웹을 앱처럼 사용 가능
   - 오프라인 지원
   - 푸시 알림

2. **React Native**
   - 네이티브 앱 경험
   - iOS/Android 동시 개발
   - 기존 React 컴포넌트 재사용 가능

3. **Expo**
   - React Native 기반
   - 간편한 배포
   - 앱 스토어 배포 지원

## 🔧 개발 환경 설정

```bash
# 1. 의존성 설치
npm install

# 2. 환경 변수 설정
cp .env.example .env.local
# .env.local 파일을 편집하여 필요한 값 입력

# 3. 데이터베이스 초기화
npm run db:generate
npm run db:push

# 4. 개발 서버 실행
npm run dev
```

## 📚 기술 스택

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: Prisma ORM (현재 SQLite, 프로덕션은 PostgreSQL 권장)
- **Authentication**: NextAuth.js
- **Deployment**: Vercel
- **Blockchain**: Saseul (향후 통합)
