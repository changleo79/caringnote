# 🔍 마이그레이션 상태 확인 방법

## 현재 마이그레이션 상태 확인

다음 방법으로 마이그레이션이 완료되었는지 확인할 수 있습니다.

---

## 방법 1: Prisma Studio로 확인

```bash
npm run db:studio
```

브라우저에서 다음 테이블이 보이면 마이그레이션 완료:
- ✅ `ResidentFamily`
- ✅ `Notification`

---

## 방법 2: 데이터베이스 직접 확인

PostgreSQL에 접속하여:

```sql
-- 테이블 목록 확인
SELECT table_name 
FROM information_schema.tables 
WHERE table_schema = 'public' 
AND table_name IN ('ResidentFamily', 'Notification');

-- ResidentFamily 테이블 구조 확인
\d "ResidentFamily"

-- Notification 테이블 구조 확인
\d "Notification"
```

---

## 방법 3: 앱에서 확인

마이그레이션이 완료되면 다음 기능이 작동합니다:

1. **프로필 수정 페이지** (`/profile/edit`)
   - 정상 작동하면 마이그레이션 완료

2. **입소자 관리 페이지** (`/residents`)
   - 페이지가 로드되면 마이그레이션 완료

3. **알림 페이지** (`/notifications`)
   - 페이지가 로드되면 마이그레이션 완료

---

## ⚠️ 마이그레이션 안 된 경우 오류 메시지

다음 오류가 발생하면 마이그레이션이 필요한 것입니다:

```
Error: 
  Invalid `prisma.residentFamily.findMany()` invocation
  ...
  Unknown model `ResidentFamily`

또는

Error: 
  Invalid `prisma.notification.findMany()` invocation
  ...
  Unknown model `Notification`
```

이 경우 마이그레이션을 실행해야 합니다.

---

## 🚀 마이그레이션 실행 명령어

```bash
npm run db:generate
npm run db:push
```
