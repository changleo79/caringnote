# 🎯 데이터베이스 연결 - 간단 해결법

## 현재 상황
이미지에서 보이는 오류: **"이미 연결되어 있습니다"**

## ✅ 가장 쉬운 해결 방법 (3단계)

### 1️⃣ 모달 창 닫기
- **"Cancel"** 버튼 클릭 (오류가 나는 모달 창 닫기)

### 2️⃣ 환경 변수 직접 추가
1. 상단 탭에서 **"Settings"** 클릭
2. 왼쪽 메뉴에서 **"Environment Variables"** 클릭
3. **"Add New"** 버튼 클릭
4. 입력:
   - **Key**: `DATABASE_URL`
   - **Value**: Storage 페이지에서 복사한 Connection String
   - **Environment**: 모두 체크 (Production, Preview, Development)
5. **"Save"** 클릭

**Connection String 찾는 법:**
- Storage 탭 → 데이터베이스 클릭 → Settings/Connection Info → Connection String 복사

### 3️⃣ 재배포
1. **"Deployments"** 탭 클릭
2. 가장 위 배포의 **"..."** 메뉴 → **"Redeploy"**
3. 완료 대기

---

## 🔍 확인 방법

### 환경 변수가 있는지 확인
```
Settings → Environment Variables
```
`DATABASE_URL` 또는 `POSTGRES_PRISMA_URL` 같은 게 있으면 이미 설정된 것입니다!

그 값 중 하나를 `DATABASE_URL`로 사용하거나,
없으면 Storage에서 Connection String을 직접 복사해서 추가하세요.
