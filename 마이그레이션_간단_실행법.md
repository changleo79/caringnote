# ⚡ 데이터베이스 마이그레이션 간단 실행법

## 🎯 가장 쉬운 방법

### Windows에서 실행

#### 방법 1: 배치 파일 실행 (가장 쉬움)
1. 파일 탐색기에서 `run_migration.bat` 파일을 찾습니다
2. 더블클릭하여 실행합니다
3. 자동으로 마이그레이션이 진행됩니다

#### 방법 2: PowerShell 스크립트 실행
1. PowerShell을 열고 프로젝트 폴더로 이동
2. 다음 명령어 실행:
   ```powershell
   .\run_migration.ps1
   ```

#### 방법 3: 수동 실행
PowerShell 또는 명령 프롬프트에서:

```powershell
# 1단계: Prisma 클라이언트 생성
npm run db:generate

# 2단계: 데이터베이스에 적용
npm run db:push
```

## 📋 실행 전 확인사항

1. ✅ Node.js가 설치되어 있는지 확인
   ```powershell
   node --version
   ```

2. ✅ npm이 설치되어 있는지 확인
   ```powershell
   npm --version
   ```

3. ✅ .env.local 파일에 DATABASE_URL이 설정되어 있는지 확인
   ```env
   DATABASE_URL="postgresql://..."
   ```

## ❓ 문제 해결

### "npm을 찾을 수 없습니다"

Node.js를 설치하세요:
- https://nodejs.org/ 에서 다운로드

### "DATABASE_URL이 없습니다"

`.env.local` 파일을 생성하고 데이터베이스 URL을 설정하세요.

### 스크립트 실행이 안 됩니다

PowerShell에서 실행 정책을 변경:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
