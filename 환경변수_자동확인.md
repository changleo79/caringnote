# 🔍 환경 변수 자동 확인 가이드

## 즉시 확인하기

브라우저에서 다음 URL들로 접속하여 상태 확인:

### 1. 데이터베이스 상태
```
https://caringnote.vercel.app/api/db-check
```

**확인 항목:**
- `connection.status`: "✅ 연결 성공" 여부
- `tables.CareCenter`: 0 이상이면 테이블 존재
- `tables.User`: 0 이상이면 테이블 존재

### 2. NextAuth 상태
```
https://caringnote.vercel.app/api/auth-check
```

**확인 항목:**
- `nextAuth.secret`: "✅ 설정됨" 여부
- `nextAuth.url`: URL 설정 여부
- `nextAuth.status`: "✅ 정상" 여부
- `recommendation`: 설정해야 할 항목 목록

---

## 🚨 로그인 오류 해결

### Step 1: 환경 변수 확인

`/api/auth-check` 접속 결과 확인:

**"❌ 설정 안 됨"이 보이면:**
1. Vercel 대시보드 → Settings → Environment Variables
2. 다음 변수 추가:
   - `NEXTAUTH_SECRET` (https://generate-secret.vercel.app/32)
   - `NEXTAUTH_URL` (`https://caringnote.vercel.app`)
3. 모든 환경 체크 후 Save

### Step 2: 재배포

환경 변수 추가 후 **반드시 재배포!**
