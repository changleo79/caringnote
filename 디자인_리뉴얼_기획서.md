# 🎨 전체 디자인 리뉴얼 기획서

## 📋 개요

요양원/케어 플랫폼 중 디자인이 가장 훌륭한 플랫폼들을 벤치마크하여 완전히 새로운 디자인 시스템으로 개편합니다.

### 벤치마크 플랫폼
- **CareZone** - 깔끔하고 미니멀한 디자인, 부드러운 색상
- **Honor Care** - 모던하고 세련된 UI, 신뢰감 있는 색상
- **Apple Health** - 프리미엄 미니멀리즘, 완벽한 타이포그래피
- **Notion** - 깔끔한 정보 구조, 넓은 여백

---

## 🎯 새로운 브랜드 아이덴티티

### 브랜드명 제안
- **현재**: "케어링노트" (Caring Note)
- **새로운 제안**:
  1. **케어라인** (CareLine) - 간결하고 모던
  2. **케어하트** (CareHeart) - 따뜻함 강조
  3. **케어링노트** 유지 - 하지만 시각적 아이덴티티 완전 재설계

**결정**: 기존 브랜드명 유지하되 시각적 아이덴티티 완전 재설계

```
Primary: #0EA5E9 (Sky Blue) - 신뢰감과 전문성
Secondary: #06B6D4 (Cyan) - 활기와 친근함
Accent: #F59E0B (Amber) - 따뜻함과 관심
Success: #10B981 (Emerald)
Warning: #F59E0B
Error: #EF4444
```

## 🎭 디자인 원칙

### 1. 미니멀리즘
- 불필요한 요소 제거
- 넓은 여백 활용
- 명확한 정보 계층

### 2. 친근함과 신뢰감
- 부드러운 곡선 (border-radius)
- 따뜻한 색상 톤
- 명확한 시각적 피드백

### 3. 접근성
- 충분한 색상 대비
- 명확한 인터랙션 영역
- 읽기 쉬운 타이포그래피

### 4. 모던함
- Glassmorphism 효과
- 부드러운 애니메이션
- 깔끔한 아이콘

## 🧩 컴포넌트 시스템

### Buttons
```css
Primary: Sky Blue 배경, 흰색 텍스트
Secondary: 투명 배경, Sky Blue 테두리
Tertiary: 회색 배경, 다크 텍스트
Ghost: 투명, hover 시 배경
```

### Cards
- **기본**: 흰색 배경, 부드러운 그림자
- **Elevated**: 더 강한 그림자, hover 효과
- **Outlined**: 테두리만 있는 카드

### Inputs
- **기본**: 회색 테두리, focus 시 Sky Blue
- **Filled**: 회색 배경
- **Outlined**: 투명 배경, 테두리

## 📱 페이지별 디자인 방향

### 홈페이지
- **Hero**: 대형 타이포그래피, 부드러운 그라데이션 배경
- **Features**: 카드 그리드, 아이콘 중심
- **CTA**: 명확한 액션 버튼

### 대시보드
- **Welcome Section**: 개인화된 인사말
- **Quick Actions**: 큰 아이콘 버튼
- **Content Cards**: 깔끔한 카드 레이아웃

### 내부 페이지
- **Header**: 명확한 페이지 제목
- **Content**: 넓은 여백, 읽기 쉬운 레이아웃
- **Actions**: 우측 상단 또는 하단 고정
